启动系统后，使用以下提示词测试 testData 向量库的检索能力：

```bash

export OPENAI='sk-your-api-key'
export WRITEHERE_EMBEDDING_MODEL='text-embedding-3-large'

# 测试搜索 API
curl -X POST http://localhost:5001/api/test-knowledge-base/search \
  -H "Content-Type: application/json" \
  -d '{"query": "AgC 平台如何创建智能体", "topk": 3}' | jq .
```

## 测试评估标准

### 检索质量评估

对于每个查询，评估返回结果：

| 评分 | 标准 |
|------|------|
| ⭐⭐⭐⭐⭐ | 结果完全相关，包含查询所需的全部信息 |
| ⭐⭐⭐⭐ | 结果高度相关，覆盖大部分关键信息 |
| ⭐⭐⭐ | 结果部分相关，提供一些有用信息 |
| ⭐⭐ | 结果相关性较低，只有少量有用信息 |
| ⭐ | 结果不相关或无用 |

## 故障注入测试

测试系统的鲁棒性：

1. **空查询**: `{"query": ""}`
2. **特殊字符**: `"测试@#$%^&*()_+"`
3. **超长查询**: 超过 1000 字的查询
4. **并发查询**: 同时发送多个请求
